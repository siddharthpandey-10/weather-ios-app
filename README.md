# 🌤️ Weather iOS App

A clean and modern iOS weather application built using **SwiftUI** and **MVVM architecture**.

---

## 🚀 Features

* 🔍 Search weather by city
* 🌡️ Real-time temperature updates
* 📍 Location-based weather
* ⚡ Fast and responsive UI
* 🎨 Beautiful SwiftUI design

---

## 🛠️ Tech Stack

* SwiftUI
* MVVM Architecture
* URLSession (API calls)
* Weather API
